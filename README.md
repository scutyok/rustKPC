<h1>rKPC</h1>

>An full remake of the game [Kiss: Psycho Circus: The Nightmare Child](https://en.wikipedia.org/wiki/Kiss:_Psycho_Circus:_The_Nightmare_Child) using the programming language Rust and the graphical API Vulkan.

<h2>How to use:</h2>

<ol>
  <li>Open project folder in VSC (or any IDE of your liking)</li>
  <li>Add the textures and the world folder from the .REZ file in the "src" project folder</li>
  <li>Name them "Textures" and "Worlds" respectively</li>
  <li>Compile with "cargo run"</li>
  <li>Wait until all the textures loads</li>
  <li>You can now see the map</li>
</ol>

<h2>Keybinds</h2>

| Key           | Function        |
| ------------- | --------------- |
| W             | Move forward    |
| S             | Move backwards  |
| A             | Strafe left     |
| D             | Strafe right    |
| Space         | Ascend          |
| Shift         | Descend         |
