//	MapMessage.txt
//
//	This is a list of messages that are used for any messages that are 
//	specified by a map designer in the .ed file.  Its sole purpose is
//	to ease localization so that text messages can be changed to another
//	language without recompiling maps.
//
//	Specify the message first.  If you want spaces in the message, you
//	must put the message inside of quotes.  If you want a quote to be
//	printed in your message, use \", as in the following example:
//
//	"The peanut exclaimed, \"You are a fool, Captain Spismo!\""
//
//	This will print:
//
//	The peanut exclaimed, "You are a fool, Captain Spismo!"
//
//	After the text, you can add a comment indicating the index of the
//	message for easy reference.  Not that the index you specify is
//	really based on the order of the messages in this file and has
//	nothing to do with the comments, so if you screw up the numbering
//	in the comments, you still have to specify the message by it's
//	order in this file.
